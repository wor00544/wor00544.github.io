js阻塞加载(默认)

函数

JS函数是作用域；
实参 —值—> 形参；

函数分定义和使用两个阶段;
函数创建方式有静态创建和动态创建;
可以直接在定义形参时用=定义默认值；
可以用name=name || “……”; 定义默认值 （es5）；
return 返回结果 （代表函数运行结束了，下面的代码不会运行),后面只能放一个参数、表达式;
必须写函数声明，执行不写不会报错但声明就无效了；
JS中函数声明和函数执行顺序没有先后之分（浏览器预编译：把变量声明和函数声明先放在上面）；
函数声明名字一样的谁最后声明就用谁的；
函数放在if、for里也会遵循if和for条件；
函数里面可以嵌套函数；
函数外声明的变量 全局变量，在函数内定义的变量局部变量；
当有局部变量和全局变量同名时，就近原则优先使用局部变量；
闭包：在函数里定义函数，拥有全局变量的生命周期；

原型、原型链、继承

当前对象：本地属性、私有属性；
原型对象：原型属性、公共属性；
本地属性被修改，只有当前对象的该属性被修改；
原型属性被修改，只有对象的该属性被修改；

继承的原则：1.本地有的用本地；2.本地没有的找原型；3.原型没有的找原型对象的原型；（原型链）

对象如果修改属性的时候没有这条属性，自动增加；
删除：delete加属性名，并不是完全删除而是将他们的联系断开，删除后的属性还存在；

闭包

三要素：嵌套结构的函数、内部函数访问了外部函数的变量、在外部函数的外面，调用内部函数；
概念：闭包是由函数以及创建该函数的词法环境组合而成。这个环境包含了这个闭包穿件时所能访问的所有局部变量;
闭包对象之间互相独立；
作用：1.让局部变量变成私有化的长存变量；2.给事件调用函数传递参数；

常见错误 ：循环内使用闭包函数；
解决方法：1.使用立即执行函数；2.使用多个闭包函数；3.ES6

错误处理、错误类型

错误类型：
Error -> 基本类型，其他错误类型继承该类型，较少见；
EvalError -> 在使用eval()函数而发生异常时触发；
RangeError -> 数值超过相应范围时触发；
ReferenceError -> 找不到对象的情况下触发；
SyntaxError -> 语法类型错误；
TypeError -> 在变量中保存着意外的类型，或者访问不存在的方法 (归根结底是变量的类型不符合要求)；
URLError -> 在使用encodeURL()、decodeURL()，URL格式不正确导致；
